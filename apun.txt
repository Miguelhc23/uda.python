Para diseñar las tablas de MySQL que corresponden al Diagrama Entidad-Relación Extendido que has proporcionado, se pueden seguir las siguientes consideraciones:

Se creará una tabla para cada entidad principal en el modelo, incluyendo los atributos relevantes para esa entidad.
Cada tabla tendrá una clave primaria, que será única para cada registro en esa tabla. Esta clave se utilizará para establecer las relaciones entre las tablas.
Las relaciones entre las entidades se representarán mediante claves foráneas, que serán utilizadas para conectar las tablas.
Con estas consideraciones en mente, se pueden crear las siguientes tablas:

Soldado:

id_soldado (clave primaria)
nombre
apellidos
grado
id_compania (clave foránea)
id_cuerpo (clave foránea)
Cuartel:

id_cuartel (clave primaria)
nombre
ubicacion
Compania:

id_compania (clave primaria)
num_actividad
id_cuartel (clave foránea)
Cuerpo:

id_cuerpo (clave primaria)
denominacion
Servicio:

id_servicio (clave primaria)
actividad
Soldado_Servicio:

id_soldado (clave foránea)
id_servicio (clave foránea)
fecha_realizacion
En la tabla Soldado, se incluyen los atributos relevantes para los soldados, y se establecen claves foráneas para relacionarlos con las compañías y cuerpos a los que pertenecen.

En la tabla Cuartel, se incluyen los atributos relevantes para los cuarteles.

En la tabla Compania, se incluyen los atributos relevantes para las compañías, y se establece una clave foránea para relacionarlas con los cuarteles en los que están ubicadas.

En la tabla Cuerpo, se incluyen los atributos relevantes para los cuerpos del ejército.

En la tabla Servicio, se incluyen los atributos relevantes para los servicios que pueden realizar los soldados.

En la tabla Soldado_Servicio, se establece una relación muchos a muchos entre los soldados y los servicios que realizan, utilizando claves foráneas para relacionarlos y una fecha de realización para identificar cada registro de la relación.

Con estas tablas, se puede representar adecuadamente la información del modelo Entidad-Relación Extendido que se ha proporcionado.
